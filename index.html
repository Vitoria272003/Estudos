Questão 1: Explique qual é a função de um servidor web em um ambiente de hospedagem e cite pelo menos um exemplo.

Resposta: O servidor web entrega arquivos estáticos de um site para o usuário. Ex: Apache.
Questão 2: Qual a principal característica do NGINX e por que ele é muito utilizado em projetos de alto desempenho?

Resposta: A principal característica é trabalhar de forma eficiente com conexões simultâneas, utilizando uma quantidade mínima de recursos do servidor. Por esse motivo é utilizado em projetos de alto desempenho.
Questão 3: Diferencie os protocolos HTTP e HTTPS. Por que o HTTPS é recomendado para aplicações modernas?

Resposta: HTTP permite a troca de informações entre cliente e servidor, mas não criptografa os dados, diferente do HTTPS que utiliza criptografia SSL/TLS para proteger os dados transmitidos, garantindo confidencialidade e integridade da informação.
Questão 4: Qual é a função das portas em uma rede de computadores e qual a importância da porta 443?

Resposta: As portas funcionam como pontos de acesso lógicos que permitem que diferentes serviços e aplicações se comuniquem entre si. A porta 443 é usada para o HTTPS. Ela é fundamental para a segurança na web.
Questão 5: Defina o conceito de endereço IP e explique sua importância em redes de computadores.

Resposta: Um endereço IP funciona como o "endereço residencial" de um dispositivo na rede. Ele permite que os dados enviados pela rede cheguem ao destino correto. É importante para identificação de dispositivos, roteamento de dados, comunicação entre dispositivos, geolocalização, segurança e controle.
Questão 6: Explique o que é hospedagem VPS e em quais situações ela é mais indicada.

Resposta: É um servidor virtual com mais controle e desempenho. Indicado quando precisa de hospedagem de vários sites com tráfego alto.
Questão 7: Descreva as vantagens do servidor web Caddy em relação a outros servidores tradicionais.

Resposta: Facilidade de uso, segurança embutida e configuração automática de HTTPS.
Questão 8: Qual servidor de aplicação é mais indicado para aplicações desenvolvidas na plataforma .NET (C#)? Explique o motivo.

Resposta: O Kestrel é o mais indicado por conta do seu alto desempenho, integração nativa com o .NET e otimizado para C#.
Questão 9: Descreva o funcionamento do modelo Cliente-Servidor no contexto de sistemas web.

Resposta: O cliente (como um navegador) faz um pedido a um servidor, que processa esse pedido e envia uma resposta com os dados requisitados (como uma página web). Essa troca acontece por meio do protocolo HTTP, e a conexão termina após cada interação.
Questão 10: Explique o que é o Apache Tomcat e para qual tipo de aplicação ele é mais indicado.

Resposta: O Apache Tomcat é um servidor de aplicações open-source projetado especificamente para executar aplicações web baseadas em Java.
Questão 11: Diferencie, com suas palavras, um servidor web de um servidor de aplicação, destacando suas funções principais.

Resposta: O servidor web mostra o que já está pronto, o servidor de aplicação cria o que for necessário. O servidor web lida com requisições HTTP e entrega conteúdo estático. O servidor de aplicação executa lógica de negócio e gera conteúdo dinâmico.
